# Narvi

Narvi is the ancient Roman goddess of health and well-being, known for her role in promoting public health and safety.

This project implements the mono-repo template for building full-stack apps powered by Go and Svelte.

## Folder Structure

- backend-api - Go API server
- frontend - Svelte web application
- packages - Shared packages and libraries

## How To

- Clone the repository or create a new repository from this template (https://github.com/zeusito/narvi)
- Review the README files in the `backend-api`, `frontend`, and `packages` directories for more information about each component
